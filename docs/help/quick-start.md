# 快速开始

欢迎使用 fast-md。它是一个轻量 Markdown 编辑器，适合快速记录笔记、整理文档和预览排版效果。

## 新建与打开

- 使用 `File > New File` 新建空白文档。
- 使用 `File > Open...` 打开本地 `.md` 或 `.markdown` 文件。
- 也可以把 Markdown 文件拖到窗口中打开。

## 编辑与保存

- 在正文区域直接输入 Markdown 内容。
- 使用 `File > Save` 保存当前文件。
- 使用 `File > Save As...` 将当前内容另存为新文件。

## 导出

- 使用 `File > Export as HTML...` 导出 HTML。
- 使用 `File > Export as PDF (Print)...` 导出 PDF。

## 主题

在 `View > Theme` 中选择内容主题。主题会应用到编辑区的 Markdown 排版，并在下次启动时继续使用。

## 多窗口

使用 `File > New Window` 打开一个新窗口。每个窗口都有独立的文件状态，关闭或退出时会分别确认未保存内容。
