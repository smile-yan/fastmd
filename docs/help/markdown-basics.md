# markdown入门

Markdown 用纯文本表示标题、列表、链接、代码块等常见文档结构。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 强调

```markdown
**加粗**
*斜体*
~~删除线~~
```

## 列表

```markdown
- 无序列表
- 另一个条目

1. 有序列表
2. 另一个条目
```

## 引用

```markdown
> 这是一段引用。
> 可以用于摘录、提示或说明。
```

## 链接与图片

```markdown
[链接文字](https://example.com)
![图片说明](image.png)
```

## 代码

行内代码使用反引号，例如 `const value = 1`。

代码块使用三个反引号：

````markdown
```js
console.log('hello')
```
````

## 表格

```markdown
| 名称 | 说明 |
| --- | --- |
| fast-md | Markdown 编辑器 |
```
