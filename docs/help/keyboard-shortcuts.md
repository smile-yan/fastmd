# 快捷键说明

以下快捷键按 macOS 习惯设计。

| 功能 | 快捷键 |
| --- | --- |
| 新建文件 | `Command + N` |
| 新建窗口 | `Command + Shift + N` |
| 打开文件 | `Command + O` |
| 保存 | `Command + S` |
| 另存为 | `Command + Shift + S` |
| 关闭窗口 | `Command + W` |
| 切换侧边栏 | `Command + Shift + L` |
| 进入或退出全屏 | `Control + Command + F` |
| 切换源码模式 | `Command + /` |
| 开发者工具 | `F12` |

## 编辑快捷键

| 功能 | 快捷键 |
| --- | --- |
| 撤销 | `Command + Z` |
| 重做 | `Command + Shift + Z` |
| 剪切 | `Command + X` |
| 复制 | `Command + C` |
| 粘贴 | `Command + V` |
| 全选 | `Command + A` |

## 退出应用

使用 `Command + Q` 退出应用。若存在未保存窗口，fast-md 会逐个窗口确认是否保存或放弃更改。
