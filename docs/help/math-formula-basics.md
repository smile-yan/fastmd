# 数学公式入门

fast-md 支持使用 LaTeX 风格的数学公式语法。行内公式适合嵌在正文中，块级公式适合单独展示推导过程。

## 行内公式

使用一对美元符号包裹公式：

```markdown
质能方程是 $E = mc^2$。
```

显示效果类似：质能方程是 $E = mc^2$。

## 块级公式

使用两对美元符号包裹公式：

```markdown
$$
\int_a^b f(x)\,dx = F(b) - F(a)
$$
```

## 常用写法

| 目标 | 写法 |
| --- | --- |
| 上标 | `x^2` |
| 下标 | `a_i` |
| 分数 | `\frac{a}{b}` |
| 根号 | `\sqrt{x}` |
| 求和 | `\sum_{i=1}^{n} i` |
| 积分 | `\int_a^b f(x)\,dx` |
| 希腊字母 | `\alpha`, `\beta`, `\pi` |

## 多行公式

```markdown
$$
\begin{aligned}
a^2 + b^2 &= c^2 \\
c &= \sqrt{a^2 + b^2}
\end{aligned}
$$
```

建议把复杂公式单独放在块级公式中，便于阅读和维护。
